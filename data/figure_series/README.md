# Figure series data exports

This directory stores datasets used by manuscript plots/graphs and appendix figure-series artifacts.

Regenerate figure-series exports with:

```bash
python scripts/export_figure_data.py --figures-only
```

## Mapping to manuscript figures

- Figure 4 (from Table 7) -> `figure4_adversarial.{json,csv}`
- Figure 5 (from Table 13) -> `figure5_weight_ablation.{json,csv}`
- Figure 6 (from Table 12) -> `figure6_threshold_sensitivity.{json,csv}`
- Figure 7 (from Table 5) -> `figure7_per_domain.{json,csv}`
- Figure 8 summary (from Table 2) -> `figure8_safety_distribution.json`
